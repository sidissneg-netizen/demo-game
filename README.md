# City Ruins - Demo Game 🎮

Интерактивная игра на HTML5 Canvas, вдохновленная мобильными casual-играми.

## 🎯 Особенности

- **Боевая система**: Стреляйте в врагов и зарабатывайте монеты
- **Система улучшений**: Повышайте урон, скорость и броню
- **Враги**: Динамичные враги с разным поведением
- **Визуальные эффекты**: Взрывы, анимации и частицы
- **Мобильная оптимизация**: Поддержка тачскрина и клавиатуры
- **Звуковые эффекты**: Встроенные звуки для действий (опционально)
- **UI меню**: Настройки, ежедневные награды, рейтинг

## 🎮 Управление

### На компьютере:
- **Стрелки** или **WASD** - движение
- **Пробел** - стрельба
- **Клик мышью** - стрельба

### На мобильном:
- **Свайп** - движение
- **Кнопка "Стрелять"** - стрельба
- **Кнопки внизу** - меню

## 📋 Механика игры

### Враги
- Появляются с верхней части экрана
- Двигаются вниз и по сторонам
- При попадании пули получают урон
- Дают монеты при уничтожении

### Улучшения
- **Урон** - увеличивает урон пули
- **Скорость** - ускоряет игрока
- **Броня** - снижает получаемый урон

### Монеты
- Зарабатываются за убитых врагов
- Используются для покупки улучшений
- Стоимость улучшений растет с каждым уровнем
